
1. **Создать таблицу** — создает новую таблицу с полями `id` и `name`.
2. **Редактор схемы** — открывает окно для редактирования структуры базы данных:
   - добавление и удаление столбцов;
   - переименование таблиц и столбцов;
   - изменение типов данных;
   - установка ограничений (`NOT NULL`, `UNIQUE`, `CHECK`, `FOREIGN KEY`).
3. **Конструктор SELECT** — визуальный редактор запросов для получения данных:
   - выбор таблиц и столбцов;
   - фильтрация (`WHERE`, `LIKE`, `~`, `~*`);
   - группировка (`GROUP BY`) и фильтрация агрегатов (`HAVING`);
   - сортировка (`ORDER BY`);
   - объединения (`JOIN`).
4. **Строковые функции** — окно для выполнения преобразований строк:
   - `UPPER`, `LOWER`, `TRIM`, `SUBSTRING`, `LPAD`, `RPAD`, `CONCAT`.
5. **Поиск** — выполняет шаблонный или регулярный поиск по таблице.
6. **Выход** — завершает работу приложения.
